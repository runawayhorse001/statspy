==============

Clone the Repository
++++++++++++++++++++

.. code-block:: bash

	git clone git@github.com:runawayhorse001/statspy.git


Install
+++++++

.. code-block:: bash

	cd statspy
	pip install -r requirements.txt 
	python setup.py install

Uninstall
+++++++++

.. code-block:: bash

	pip uninstall statspy

Test
++++

.. code-block:: bash

	cd statspy/test
	python test1.py
